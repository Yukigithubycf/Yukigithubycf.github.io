# YukiYCF Homepage

基于 Vue 3 + Vite + Vuetify 的个人主页项目。

## 开源声明

本项目参考并改编自开源项目：

- 原项目地址：https://github.com/leleo886/leleo-home-page

感谢原作者的开源与分享。本仓库在原项目基础上进行了个性化配置与内容调整，仅用于个人主页展示。

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址会在终端输出。

## 构建与预览

```bash
npm run build
npm run preview
```

## 个性化配置

主要配置文件：

- `src/config.js`

你可以在该文件中修改站点标题、头像、标签、项目卡片、壁纸、音乐等信息。

## 部署说明

项目可部署到 Vercel、Cloudflare Pages 等静态托管平台。  
每次修改后执行 `git add`、`git commit`、`git push` 即可同步到 GitHub。
